# Coding Lane

个人技术博客，记录工程实践、算法题解、音乐创作等内容。基于 GitHub Pages + Jekyll 搭建。

## [访问博客 →](https://xieyw.xyz/)

## 技术栈

- **静态站点生成器**: Jekyll
- **托管平台**: GitHub Pages
- **前端框架**: Bootstrap
- **评论系统**: Gitalk
- **搜索功能**: Simple Jekyll Search
- **图标库**: Font Awesome

## 本地开发

```bash
# 安装依赖
bundle install

# 启动本地服务器
jekyll serve

# 访问 http://127.0.0.1:4000
```
